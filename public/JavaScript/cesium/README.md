# Cesium 二次封装

## 目录 （方法文件 上方有使用方法）

```
┌─Tripartite        三方依赖库
│  ├─geojson        json转geojson
│  ├─gcoord         坐标系转换
│  ├─layui          ui
│  ├─layer          提示
│  └─uuid-js        生成唯一id
├─Map               地图
│  ├─Baidu          百度  
│  └─Gaode          高德 
├─Materials         材质
├─Config.js         配置项
├─Entity.js         实体创建
├─Primitives.js     图元
├─Utils.js          工具
├─Echarts.js        转换方法
├─Transform.js      转换方法
├─Canvas.js         原生canvas操作
├─index.js          入口文件
└─readme            自述文件夹
```
